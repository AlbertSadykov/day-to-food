# Gulp сборка для вёрстки

## Установка
```bash
npm install
```

## Разработка
```bash
npm run dev
```

## Сборка
```bash
npm run build
```

## Структура
- `src/` — исходники
- `dist/` — собранные файлы

## Что входит
- HTML копируется в `dist`
- SCSS компилируется в CSS и минифицируется
- JS собирается в один файл и минифицируется
- Картинки оптимизируются
- Шрифты копируются
- Работает live-reload через BrowserSync
